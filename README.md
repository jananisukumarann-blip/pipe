

## Project Overview

The **Premium HDPE Pipes Website** is a modern, responsive product landing page designed to showcase HDPE pipe products and their technical specifications. The website highlights product features, industry applications, manufacturing processes, resources, and customer testimonials to provide a complete overview for potential buyers and partners.

The goal of this project is to present HDPE pipe solutions in a **clear, professional, and visually appealing way** while maintaining a smooth user experience across different devices.

---

# Technologies Used

### Frontend

* **HTML5** – Used to structure the content and layout of the website.
* **CSS3** – Used for styling, layout design, responsiveness, and visual effects.
* **Bootstrap 5** – Used for responsive grid system and UI components.
* **JavaScript** – Used to add interactivity such as image carousel and FAQ accordion.
